# Jewelkunst

Premium-Schmuckmarke mit Fokus auf leise Eleganz, Bedeutung und zeitlose Schmuckstuecke.

## Ziel

Dieses Repository dient als zentrale Grundlage fuer die digitale Jewelkunst-Praesenz:

- Landingpage
- Markenbeschreibung
- SEO-Grundlage
- GitHub-Pages-Deployment
- spaetere Produkt-, Kampagnen- und Automationslogik

## Markenwelt

Jewelkunst steht fuer Schmuck, der nicht laut sein muss. Warme Materialien, ruhige Formen und eine persoenliche Ausstrahlung fuer Halsketten, Armbaender, Ohrringe und Ringe.

## Deployment

GitHub Pages wird ueber Actions deployed. In GitHub unter **Settings > Pages** muss als Source **GitHub Actions** gewaehlt sein.

Letzter Deployment-Trigger: README aktualisiert, um den GitHub-Pages-Workflow erneut auszufuehren.
